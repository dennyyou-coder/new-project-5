A few days ago, a friend sent me a robot vacuum product on Kickstarter and said it had many highlights.


The slogan said Narwal is the world's first robot mop and vacuum featuring innovative self-cleaning technology.

![Narwal Robot Vacuum Innovation image 2](/images/insights/narwal-robot-vacuum-innovation-image-002.jpg)

![Narwal Robot Vacuum Innovation image 3](/images/insights/narwal-robot-vacuum-innovation-image-003.jpg)

The core self-cleaning function is achieved through two water tanks: one dirty-water tank and one clean-water tank. The robot then runs automatically on the base. Raised scraping grooves on the tray contact the mop and clean off dirt from it.

![Narwal Robot Vacuum Innovation image 4](/images/insights/narwal-robot-vacuum-innovation-image-004.jpg)

![Narwal Robot Vacuum Innovation image 5](/images/insights/narwal-robot-vacuum-innovation-image-005.jpg)

![Narwal Robot Vacuum Innovation image 6](/images/insights/narwal-robot-vacuum-innovation-image-006.jpg)

At the same time, the cleaning effect of the automatically rotating mop is much stronger than that of common mopping robots currently on the market. Dried red-wine stains and marker marks can all be cleaned off.

![Narwal Robot Vacuum Innovation image 7](/images/insights/narwal-robot-vacuum-innovation-image-007.jpg)

It is also equipped with an 18,000 Pa motor, vacuuming while mopping.

![Narwal Robot Vacuum Innovation image 8](/images/insights/narwal-robot-vacuum-innovation-image-008.jpg)

It uses a LiDAR navigation solution.

![Narwal Robot Vacuum Innovation image 9](/images/insights/narwal-robot-vacuum-innovation-image-009.jpg)

Virtual walls can be set to prevent it from getting stuck in places such as carpets.

![Narwal Robot Vacuum Innovation image 10](/images/insights/narwal-robot-vacuum-innovation-image-010.jpg)

Noise is as low as 45 dB.

![Narwal Robot Vacuum Innovation image 11](/images/insights/narwal-robot-vacuum-innovation-image-011.jpg)

The overall appearance is shown below. One can see that the design is very simple, comparable with Xiaomi ecosystem design.

![Narwal Robot Vacuum Innovation image 12](/images/insights/narwal-robot-vacuum-innovation-image-012.jpg)

The crowdfunding price is USD 549. The final selling price of this product should also be relatively high.

![Narwal Robot Vacuum Innovation image 13](/images/insights/narwal-robot-vacuum-innovation-image-013.jpg)

![Narwal Robot Vacuum Innovation image 14](/images/insights/narwal-robot-vacuum-innovation-image-014.jpg)

By now it has already raised USD 780,000 through crowdfunding.
