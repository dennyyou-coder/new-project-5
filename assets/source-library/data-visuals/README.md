# World Clean Biz Article Data Visual Library

Template status:

```text
Template E: Data Visual
Status: Formalized in Visual System V1.1
```

This folder tracks data visuals generated from World Clean Biz article content.

The purpose is different from the source image library:

```text
Source asset library: real logos, people, products, events, factories.
Article data visual library: WCB-generated charts, data cards, market maps, brand matrices, comparison tables.
```

## Rule

Final market/data images should be generated from the data used inside the article.

Annual reports, investor presentations, prospectuses, SEC filings, and IR pages are verification sources only. They are not the default image material.

## Template E: Data Visual

Use this template for:

```text
revenue change
market share
brand split
growth decomposition
regional split
channel split
category mix
product mix evolution
risk / debt stack
```

Core layout:

```text
Title
Chart
Key Insight
Key Numbers
Source / unit label
```

The key insight should be data-specific, not slogan-like.

Prefer:

```text
Ninja contributed 67% of total growth between 2022 and 2024.
```

Avoid:

```text
Ninja became the main growth engine.
```
