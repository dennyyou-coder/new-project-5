## Operating Rule

Do not replace every image.

Upgrade an image only when it improves:

```text

Judgment
Evidence
Reading completion
```

## Production Workflow Per Batch

1. Read each article and extract article-native data.
2. Check available source assets.
3. Decide per article:
   - cover only
   - cover + one E data visual
   - body evidence replacement
   - hold
4. Generate candidate images into:

```text
assets/source-library/article-replacements/<batch-name>/
```

5. Create a review sheet:

```text
assets/source-library/_review/<batch-name>-review.jpg
```

6. Apply only approved images to:

```text
public/images/insights/
content/insights/*.mdx
```

7. Run build.
8. User reviews local / production links.
9. Commit and push only website-facing files.

## Variation Rule

For multiple articles about the same brand or category, covers must vary by structure:

```text
Brand logo + product hero
Brand logo + data chart
Market matrix
Risk stack
Timeline / milestone ladder
Channel map
Capital / ownership structure
```

Do not repeat the same left-title/right-product layout across nearby articles.

## Template Rotation Rule

Future batches should rotate the five approved WCB templates instead of defaulting to data cards.

Use:

```text
A Brand Analysis
B Market Analysis
C Founder Profile
D Industry Opinion
E Data Visual
```

Batch-level rule:

```text
Each batch should include at least two to three different templates when the article mix allows it.
```

Practical mapping:

| Template | Priority Visual Asset |
|---|---|
| A Brand Analysis | Real brand logo + real product / company evidence |
| B Market Analysis | Market matrix + brand map + category structure |
| C Founder Profile | Confirmed founder / CEO portrait + logo + one data anchor |
| D Industry Opinion | Real event / forum / factory / supply-chain / channel scene |
| E Data Visual | Article-native data chart with key insight |

Important:

```text
Do not force Template C without a confirmed person identity.
Do not force Template E when article numbers are inconsistent or uncertain.
When a cover feels too plain, increase the role of real product or evidence photography before adding decoration.
```
