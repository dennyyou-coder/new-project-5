# World Clean Biz Source Asset Library

This folder stores reviewed source assets for World Clean Biz covers, body evidence images, reports, and article visuals.

The visual templates are now stable. The next constraint is source material quality.

## Asset Priorities

Build the library in this order:

```text
1. Top brand logos
2. Founder / CEO / executive photos
3. Product photos
4. Forum / conference / event photos
5. Exhibition photos
6. Factory / supplier visit photos
7. Article-data chart inputs
8. Clean internal infographics generated from article data
```

## Target Coverage

Initial target:

```text
Top 50 brand logos

Top 50 product images
Forum photo library
Exhibition photo library
Factory visit photo library
Article-data chart library
```

```text

  logos/

```text

  forums/
  exhibitions/

  market-data/
```

## Review Rules

Every source asset must be reviewed before being used on the website.

Record:

```text
Asset path
Source URL or source note
Company / person / product identity
Usage rights / source type
Reviewer decision
Approved use cases
```

## Market Data Image Rule

Market/data images should be generated from the data already used in World Clean Biz articles.

Do not treat annual report, investor presentation, prospectus, SEC, or IR screenshots as the default image source.

Use those documents only to verify the numbers, dates, company names, and source labels behind charts.

For final images, prefer:

```text
WCB-generated data cards
WCB-generated charts
WCB-generated market maps
WCB-generated comparison tables
WCB-generated brand matrices
```
```

Avoid:

```text
Unverified portraits
Generic AI images
Low-resolution logos
Product images with unclear brand identity
Decorative images without information value
```
